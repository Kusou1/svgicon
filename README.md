# Svg Icon Packages

## Features
- Easy to use svg as icon in vue, react or other UI framework.
- Support multiple colors.
- Support use fill and stroke at the same time.
- Svg icon preview

## Packages
- @yzfe/vue-svgicon
- @yzfe/react-svgicon
- @yzfe/svgicon-loader
- @yzfe/svgicon
- @yzfe/svgicon-gen
- @yzfe/svgicon-viewer
- @yzfe/svgicon-polyfill
